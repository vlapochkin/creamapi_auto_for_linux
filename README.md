# CreamAPI Auto for Linux (SmokeAPI Integration)

Автоматизированный инструмент для установки и настройки **SmokeAPI/CreamAPI** в играх Steam на Linux. 

Проект создан для упрощения процесса разблокировки DLC, автоматически определяя архитектуру игры (32/64 bit), тип запуска (Native Linux или Proton) и прописывая необходимые параметры окружения.

## Особенности
- 🐧 Поддержка Native Linux игр (`libsteam_api.so`).
- ⚛️ Поддержка Windows игр через Proton (`steam_api.dll`, `steam_api64.dll`).
- 🛠 Автоматическое создание `SmokeAPI.config.json` с нужным AppID.
- 📋 Авто-копирование параметров запуска для Proton (`WINEDLLOVERRIDES`).
- 🔄 Встроенный загрузчик актуальных версий ядра SmokeAPI с GitHub.
- 🎨 Современный интерфейс на GTK4 + Libadwaita.

## Использование (AppImage)
1. Скачайте последнюю версию в разделе **Releases**.
2. Дайте файлу права на исполнение: `chmod +x cream-api-auto-x86_64.AppImage`.
3. Запустите приложение. Оно само найдет ваши библиотеки Steam и список установленных игр.
